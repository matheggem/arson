ARSON — image slots
===================

These are the four photographs the site references, in scroll order:

  jonah-01.png   →  Cello bow / strings, low light
  jonah-02.png   →  Cello + hands + feet
  jonah-03.png   →  Dancers in white (indoor / double-exposed)
  jonah-04.png   →  Cloth + figures, outdoors at night

If you swap any of these out, just keep the same filename and the site picks
it up automatically. To re-order them across the page, edit the data-src=""
attributes on the <div class="img-frame"> elements in index.html.

Future drop-in slots (already wired in CSS / JS):
